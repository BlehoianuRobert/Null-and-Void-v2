The "Smart Hat" acts as a wearable radar. The HC-SR04 ultrasonic sensor measures the distance to obstacles in front of the user. The ESP32-C6 processes this data and triggers a Buzzer with increasing frequency as the user gets closer to an object (similar to a car's parking sensor).
2. Hardware Requirements

Microcontroller: ESP32-C6 (Development Board)



Sensor: HC-SR04 Ultrasonic Distance Sensor



Output: Active Piezo Buzzer (3.3V or 5V compatible)



Power: 3.7V LiPo Battery or a small Power Bank



Misc: Jumper wires, a hat, and a breadboard or perf-board.
